
## Fix me a Burger - Online ordering site.

A web and mobile application for online ordering which gives the user the avility to construct their own hamburger. 

### Main components:
* Burger builder
* Authentification, Sign-in and Sign-up form 
* User order log
* Order summary
* Order checkout
* Contact data form

## Build with
* React & react-router
* Redux & redux-thunk
* axios
* Firebase

## Testet with:
* Jest.js
* Enzyme
